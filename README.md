# Principles of Object-Oriented Programming

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/from-referrer/)

_Principles of Object-Oriented Programming_ is a textbook published by [OpenStax](https://openstax.org/), a non profit organization that is part of [Rice University](https://www.rice.edu/).

The book can be viewed [online](https://github.com/cnx-user-books/cnxbook-principles-of-object-oriented-programming/releases/latest), where you can also see a list of contributors.
